    Циклусот for (for loop)

    Наредбата for се користи за итерирање низ елементите на некоја последица (како што се низа, кортеж или листа) или некој друг објект што може да биде итериран. Последицата се вреднува само еднаш. При секоја итерација, променливата дефинирана во циклусот for ќе биде присвоена на следната вредност во листата. Кодот што следи после линијата со наредбата for се извршува еднаш за секоја ставка. Кога ставките се искористени, циклусот завршува. 

    
      #Испечатете го секој прост број од простите список користејќи ја јамката for.
          Прост број е природен број поголем од 1 кој нема позитивни делители освен 1 и самиот себе.
      for i in range(5):    # За секој број i во опсегот од 0 до 4. Функцијата range(5) враќа листа [0, 1, 2, 3, 4]
      print(i)          # Оваа линија се извршува 5 пати. Прв пат i е 0, потоа 1, ...

      primes = [2, 3, 5, 7]   # Креирајте нова листа.

    Листа за собирање

    Можете да користите циклус за изградба на листа (или друга структура на податоци). На пример: 

    
      my_list = []
      for i in range(5):
      my_list.append(i)

      print(my_list)

    Излез: 
    
     [0, 1, 2, 3, 4]
    Ова е убаво, но доста гломазно. Разбирањето на списоци нуди покомпактна синтакса кога сакате да креирате нова листа врз основа на вредностите на постоечката листа или друга повторувачка (дупла, низа, низа, опсег, итн.). Ја прави истата задача и ја поедноставува програмата. Вообичаено, разбирањата на листите се напишани во една линија код. 
    
      my_list = [i for i in range(5)]
      print(my_list)
    Излез: 
    
      [0, 1, 2, 3, 4]

    While циклусот

    While циклусот е малку сличен на if наредбата: извршува одреден код ако одреден услов е вистински. Клучната разлика е што ќе продолжи да го извршува внатрешниот код се додека условот е вистински. Ако изразот е лажен, циклусот се прекинува. 

    
      #Испечати ги сите квадрати од 1 до 9 (1, 4, ..., 81). Користи променливата number во while циклусот.
      square =  1   # Воведена е променлива за следење на повторувањата

      while square <=  10:
      print(square)     # Овој код се извршува 10 пати
      square +=  1       # Овој код се извршува 10 пати

      print( "Finished")   # Овој код се извршува еднаш

      square =  0
      number =  1

      # Испечатете ги сите квадрати од 1 до 81.
      # Користете изјава за while и услов овде.
      square = number **  2
      print(square)
      number +=  1

    Else со циклуси

    Се видовме дека наредбата break го прекинува највнатрешниот зацврстен for или while циклус.
    Во Python исто така се дозволува на циклусните наредби да имаат клазула else. Таа се извршува кога циклусот завршува преку исцрпување на итерабилниот објект (со for) или кога условот станува Лажен (со while), но не се извршува кога циклусот се прекинува со наредбата break. Погледнете го следниот пример на циклус кој бара прости броеви: 

    
      for n in range(2, 10):
      for x in range(2, n):
      if n % x == 0:
      print(n, 'equals', x, '*', n//x)
      break
      else:
      #Циклусот заврши без да најде делител.
      print(n, 'is a prime number')

    Клучниот збор "continue" (продолжи)

Циклусот се враќа на for или while наредбата и повторно ја оценува условот. Ако нема следен елемент, се извршува делот "else" ако постои.

      print(i)
