    Булови оператори

    Буловите оператори споредуваат изјави и враќаат вредности во булови вредности. Буловиот оператор "and" враќа True кога изразите на двете страни на "and" се True. Буловиот оператор "or" враќа True кога барем еден израз на едната или другата страна на "or" е True. Буловиот оператор "not" инвертира буловскиот израз пред него. 

    
       #Напишете израз кој ќе се врати вистинит ако името е еднакво на "John" и ако тој има 16 години или повеќе.
      name = "John"
      age = 17

      print(name == "John" or name == "Jane")    # Проверете дали името е еднакво на "John" до "Jane"

    Редослед на булови оператори

    Буловите оператори не се вреднуваат од лево кон десно. Постои редослед на операции за буловите оператори: прво се вреднува операторот "not", потоа операторот "and", а на крај операторот "or". 

    
      #Напишете израз кој ќе се врати вистинит ако името е или "John" или "Jane", и ако тие се на возраст од 16 години или повеќе, но помлади од 25.
      name = "John"
      age = 17

      print(name == "John" or not age > 17)

      print(name == "John" and not age == 17)

    If наредба

    Сложените наредби во Python содржат (групи од) други наредби; тие влијаат или контролираат извршувањето на тие други наредби на некој начин.
    Најпознатиот вид на наредба е if наредбата. Клучниот збор if се користи за формирање условна наредба која извршува одреден код откако ќе го провери дали изразот е True. Python користи вметнатост за дефинирање на групи на код: 

    
      if value > 1000:
      print("It's a large number!")  # Вовлечен блок
      a += 1                         # Вовлечен блок

      print("Outside the block!")

    Блокот на код започнува со вметнување и завршува со првата линија која не е вметната. Количината на вметнувањето мора да биде консистентна во целиот блок. Обично се користат четири празнини или еден таб за вметнување. Неправилно вметнување ќе резултира со IndentationError (Грешка во вметнувањето). Ако имате само една наредба за извршување, можете да ја ставите на истата линија како и if наредбата. 
    
      if a > b: print("a is greater than b")

    Наредбите elif и else

    Наредбите elif и else дополнуваат if наредбата. Може да има нула или повеќе делови elif, а делот else е опционален. Клучниот збор elif е скратено од 'else if' и е корисен за избегнување на прекумерно вметнување.
    Во условното извршување, само еден од блоковите се избира со тоа што се вреднуваат изразите еден по еден сè додека не се пронајде израз кој е True. Тогаш тој блок се извршува и ниту еден друг дел од if наредбата не се вреднува. Ако сите изрази се лажни, се извршува блокот од делот else, ако постои. 

    
      a = 200
      b = 33
      if b > a:
      print("b is greater than a")
      elif a == b:
      print("a and b are equal")
      else:
      print("a is greater than b")

      if a > b:

    Едно-линиска if-else наредба

    Како што видовте во претходните задачи, едноставната if-else наредба може фактички да се напише како една линија. Препишете ја наредбата во кодниот едитор така што да се вклопува во една линија, но резултатот од нејзиното извршување да остане точно истиот. 

    
      from random import random  # Внесување на модул за псевдо-случаен генератор на броеви.

      my_random_number = random() * 100

      if my_random_number > 50:
      print(my_random_number)
      else:
      print("Too small!")
